Glossário

Variáveis servem para armazenar informações.

let → valor pode mudar

const → valor não muda

let nome = "Neidinha";
const idade = 30;

alert → mostra mensagem

prompt → recebe dado do usuário

alert("Olá!");
let nome = prompt("Digite seu nome:");

Executa algo se a condição for verdadeira.

if (idade >= 18) {
    alert("Maior de idade");
}

Compara valor e tipo.

5 === "5" → false
5 === 5 → true

Bloco de código reutilizável.

function saudacao() {
    alert("Olá!");
}

Permite manipular elementos da página.

document.getElementById("titulo").innerHTML = "Novo texto";

Executa algo quando o usuário interage.
